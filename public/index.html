货币政策（影响力：极高）

美联储的政策决定是影响美股最直接、最强力的单一因素

核心逻辑：利率上升 → 企业融资贵、估值压缩、债券吸引力增加 → 股市承压；利率下降 → 流动性宽松、估值扩张 → 股市受益。
🎯

FOMC 利率决议

每年召开 8次 会议，决定联邦基金利率目标区间。加息/降息/维持均会引发大盘剧烈波动，是全年最重要的单一事件。

影响力
极高
每年8次加息利空降息利好
🎤

鲍威尔新闻发布会

每次FOMC后召开，"鹰派/鸽派"措辞可在 30分钟内 引发指数1-3%的波动。是最受市场关注的单一讲话事件。

影响力
极高
实时波动鹰派利空鸽派利好
📋

FOMC 会议纪要

会议结束约 3周后 公布，揭示委员们的分歧与讨论细节，市场从中寻找未来政策路径线索，常引发二次波动。

影响力
较高
会后3周政策预期
🏦

QE / QT 资产负债表

量化宽松（QE）扩表注入流动性，利好股市；量化紧缩（QT）缩表抽走流动性，压制股市。2022年QT是当年熊市的重要推手。

影响力
高
流动性QT利空QE利好
📈

关键经济数据（影响力：极高～高）

定期发布的数据直接影响美联储政策预期，进而驱动大盘走势

🌡️

CPI — 消费者价格指数

衡量通胀的核心指标，每月第2周发布。高于预期 → 推迟降息 → 股市下跌；低于预期 → 降息预期升温 → 股市上涨。

影响力
极高
每月第2周高于预期利空低于预期利好
👷

NFP — 非农就业数据

每月第一个周五发布，是美联储"就业最大化"目标的核心参考。就业强劲但可能推迟降息；就业疲软则降息预期增强。

影响力
极高
每月第一周五双向解读
🛒

PCE — 个人消费支出价格

美联储最偏好的通胀指标（优先于CPI），每月底发布。核心PCE（剔除食品和能源）是政策制定的关键参考。

影响力
高
美联储首选月底发布
📊

GDP — 国内生产总值

每季度发布初值、修正值和终值。连续两季度负增长即为技术性衰退，会引发大盘大幅下跌。

影响力
高
季度发布衰退信号
🏭

PPI — 生产者价格指数

衡量生产端通胀，是CPI的先行指标。每月第2周（CPI前一天）发布，上涨往往预示未来CPI也将上涨。

影响力
较高
CPI先行指标生产端通胀
🛍️

零售销售 / PMI / 消费信心

零售销售衡量消费支出（占GDP约70%）；PMI反映制造业/服务业景气度（50为荣枯线）；消费信心指数反映消费预期。

影响力
中高
每月消费/景气
🌍

地缘政治与其他因素

非经济因素同样能在短时间内引发大盘剧烈波动

📊

10年期美债收益率

无风险利率基准，直接影响股票估值（DCF折现率）。收益率上升 → 估值压缩，尤其打压高估值成长股。

影响力
极高
收益率↑利空收益率↓利好
⚔️

地缘政治冲突 / 中美关系

重大军事冲突引发避险情绪；中美贸易战、科技脱钩、关税加征对科技和半导体板块冲击尤为显著。

影响力
高
短期剧烈避险情绪
😱

VIX 恐慌指数 / 黑天鹅

VIX > 30为极度恐慌信号。疫情、金融危机等黑天鹅事件短期冲击极端（2020年3月S&P 500单月跌34%），但历史上均最终恢复。

影响力
极端

金融板块财报（财报季开幕信号）

大型银行财报通常是每个财报季的"第一枪"，反映整体经济健康度

🏦

摩根大通（JPMorgan Chase）

美国最大银行，通常是财报季第一家发布财报的大型金融机构。净息差（NIM）、贷款增长和信贷损失拨备是关键指标，反映整体经济信用状况。

财报季开幕经济信用风向
🏦

高盛 / 摩根士丹利

投资银行业务（IPO、并购、债券承销）反映资本市场活跃度。交易收入波动大，财报超预期往往提振金融板块整体情绪。

资本市场金融板块
🛒
